Wedding HQ Studio

Quick Start Guide — Browser Edition

What You Have

Wedding HQ Studio is a self-contained browser-based wedding planner. It runs entirely inside a single HTML file — no internet required after opening, no login, no subscription. Everything you enter is saved automatically to your browser.

How to Open It

Step 1
Find your HTML file

Locate the Wedding-HQ-Studio-[Edition].html file from your download.

Step 2
Open in your browser

Double-click the file, or right-click → Open With → Chrome, Safari, Firefox, or Edge. Any modern browser works on Mac, Windows, iPad, or Android.

Step 3
Complete the setup flow

Enter your names, wedding date, venue, and budget goal. Choose your color palette. Click Create Planner — you're in.

The planner saves automatically every time you make a change. You do not need to press save.

Switching Devices

Because your data is stored in the browser (not the cloud), it does not automatically sync between devices. To move your planner to another device:

  1. Go to Settings → Export JSON Backup
  2. Save the downloaded .json file to cloud storage (Google Drive, iCloud, Dropbox)
  3. On your other device, open the HTML file in a browser
  4. Go to Settings → Restore from JSON and select your backup file
Export a JSON backup regularly — especially before clearing your browser history or switching devices. It takes 5 seconds and keeps your data safe.

Using on iPad

Open the HTML file in Safari or Chrome on iPad. For the best experience:

  • In Safari, tap the Share button → Add to Home Screen for an app-like icon
  • Use landscape orientation for the full sidebar layout
  • The planner is fully touch-friendly — no Apple Pencil required

Dark Mode

Toggle dark mode using the switch at the bottom of the left sidebar, or in Settings → Appearance. Your preference is remembered automatically.

Color Palettes

Choose from four palettes during setup or change anytime in Settings:

  • Editorial — Cream, taupe, muted olive (default)
  • Soft Blush — Warm rose neutrals
  • Modern Navy — Crisp cool luxury
  • Warm Terracotta — Earthy editorial tones

Printing Your Planner

Go to Settings → Print Planner. Your browser's print dialog will open with print-optimized styles — sidebars and buttons are hidden, content flows cleanly to paper or PDF.

Resetting or Starting Over

Go to Settings → Danger Zone → Reset All Data. This permanently clears all saved data from the browser. Export a JSON backup first if you want to keep anything.
